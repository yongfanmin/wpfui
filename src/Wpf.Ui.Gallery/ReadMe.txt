软件流程

/类库
:获取唯一机器码
:http请求类
:文件下载类
:运行环境信息获取
:配置文件校验

/配置文件
:存储接口域名
:存储机器码用于校验
:存储用户名密码用于快速登录
:绑定机器

启动程序:
:上报IP和机器码
:校验配置文件(机器码/打印机是否一致)
:获取机器信息机型最优配置 (内存 CPU核心线程数 磁盘剩余空间检测 程序安装路径/文件存放路径判断)

>登录
:校验身份/生产权限

>订单抓取

>产品信息抓取(裁片/印花图/印花图位置信息/排版信息)

>裁片/印花图抓取

>印花图根据位置信息进行旋转/位移/等操作

>裁片/印花图放大(AI+NetVips) 放大倍数平衡点 1.8

>裁片/印花图叠加合成

>多裁片进行单一生产图排版/多版进行合并成单一生产图
----------------
合成图核心:
1.获取订单信息>
包含原始的 裁片图地址 印花图地址
单个裁片对应多个印花图，每个印花图再裁片上的 相对位置|旋转角度|缩放比例
裁片图 印花图 对应实际生产尺寸的缩放比例
带印花图的裁片图最终排版图
额外:多个排版图合成一个排版图
